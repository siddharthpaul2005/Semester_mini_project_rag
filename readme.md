# AI Tutor - RAG based Learning Assistant

AI Tutor is a full-stack educational web application designed to help students learn effectively through interactive document-based Q&A, automated quiz generation, and analytics. It utilizes Retrieval-Augmented Generation (RAG) to provide contextual answers from uploaded PDF study materials and evaluates student performance using advanced Language Models.

## Features

- **Document Processing**: Upload PDF study materials, which are chunked and stored in a vector database for efficient retrieval.
- **RAG-based Chat**: Ask questions about your study materials and get context-aware answers generated by a local LLM.
- **Automated Quiz Generation**: Generate topic-based quizzes to test your knowledge.
- **Answer Evaluation**: Submit quiz answers and receive intelligent feedback and scoring.
- **Analytics**: Track your performance across different topics with built-in analytics.

## Tech Stack

### Backend
- **Framework**: FastAPI (Python)
- **Vector Database**: ChromaDB
- **Embeddings**: SentenceTransformers (`BAAI/bge-small-en`)
- **LLM Engine**: Ollama (`deepseek-r1:8b`)
- **Text Processing**: LangChain (PyPDFLoader, TextSplitter)
- **Database**: SQLite

### Frontend
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **HTTP Client**: Axios

## Prerequisites

- [Python 3.9+](https://www.python.org/downloads/)
- [Node.js 18+](https://nodejs.org/)
- [Ollama](https://ollama.com/) (installed and running locally)

## Installation & Setup

### 1. Model Setup (Ollama)
First, ensure Ollama is running, then pull the required model:
```bash
ollama run deepseek-r1:8b
```

### 2. Backend Setup
Navigate to the backend directory and install dependencies:
```bash
cd backend
python -m venv .venv
# Activate venv:
# Windows: .venv\Scripts\activate
# Mac/Linux: source .venv/bin/activate
c:\AI-Tutor\.venv\Scripts\Activate.ps1 
to activate venv

pip install -r requirements.txt
```
Start the FastAPI server:
```bash
uvicorn main:app --reload --port 8000
```

### 3. Frontend Setup
Navigate to the frontend directory and install Node modules:
```bash
cd frontend
npm install
```
Start the Vite development server:
```bash
npm run dev
```

## Usage
1. Open your browser and navigate to the frontend URL (typically `http://localhost:5173`).
2. Upload a PDF document in the app.
3. Chat with the AI Tutor to ask questions about the document.
4. Generate a quiz on a topic, answer the questions, and receive feedback.
5. Check your learning progress in the Analytics section.

## Architecture
- **`backend/main.py`**: Entry point for API routes (`/upload`, `/ask`, `/generate_quiz`, `/submit_quiz`, `/analytics`).
- **`backend/rag_pipeline.py`**: Handles PDF parsing, chunking with LangChain, and ChromaDB vector indexing.
- **`backend/tutor_engine.py`**: Interacts with the RAG pipeline and Ollama to generate contextually accurate answers.
- **`backend/database/`**: Contains the SQLite database and connections for tracking quiz results.
- **`frontend/src/`**: Contains React components, pages, and API integration logic.
